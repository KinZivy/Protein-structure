# 第三章 氨基酸的肽基与稳定作用
## 3.1 肽键的形成与结构特征
### 3.1.1 肽键的生物合成与化学本质
肽键是蛋白质一级结构的基本连接单元，其形成是一个精确的酶催化过程。在核糖体中，肽基转移酶中心催化氨基与羧基间的亲核取代反应，释放一分子水形成-CO-NH-共价键。

**电子结构特性**：肽键具有显著的部分双键特征（约40%双键性质），这是由于羰基氧原子与酰胺氮原子间的共振相互作用。这种共振产生了以下关键效应：
- C-N键长缩短至1.32 Å（介于C-N单键1.47 Å和C=N双键1.27 Å之间）
- 旋转能垒增高至75-85 kJ/mol，远高于C-C单键的旋转能垒（12 kJ/mol）
- 形成刚性的平面结构，六个原子（Cα₁-C-O-N-H-Cα₂）几乎完全共面，也称肽平面

### 3.1.2 肽平面的几何约束与构象影响
肽键的平面性对多肽链构象产生严格约束。参与肽键的六个原子形成刚性平面，其平面性偏差通常小于0.03 Å（Berkholz et al., 2020）。这种几何约束导致：
- 顺反异构：99.6%的肽键采取反式构型（羰基氧与酰胺氢相对），仅约0.4%（主要发生在脯氨酸前）为顺式构型，因脯氨酸的环状侧链减小了立体冲突。
- 二面角限制：多肽链的构象自由度主要体现在φ（C-N-Cα-C）和ψ（N-Cα-C-N）两个旋转角上。这些角度的允许组合受立体阻碍限制，可用拉氏构象图系统描述。
