# 第二章 氨基酸序列与结构稳定作用

## 2.8 蛋白质一级结构
蛋白质一级结构是最基础、最具决定性的结构层次，也是结构蛋白质组学中序列 — 结构 — 功能研究链条的逻辑起点。

一级结构的严格定义为：多肽链中氨基酸残基的线性排列顺序。氨基酸之间通过肽键共价相连，形成连续、有固定方向的多肽主链，这一序列是蛋白质所有高级空间结构形成的唯一化学信息基础。

在表示与书写上，蛋白质一级结构通用三字母缩写和单字母缩写两套标准体系，是序列比对、数据库存储、结构建模的通用语言。书写与阅读均严格遵循N‑端 → C‑端的方向，与体内核糖体合成多肽链的方向一致。

结构生物学的核心基石Anfinsen 原理明确指出：一级结构决定高级结构。在合适的环境中，多肽链仅依靠自身氨基酸序列信息，即可自发折叠成具有生物活性的天然三维结构。这一原理从根本上确立了一级结构在蛋白质结构形成中的支配地位。
在结构蛋白质组学中，一级结构是整个研究的入口：
以一级序列为基础，可进行保守区、结构域、功能基序的识别；
依托一级序列开展蛋白质结构预测（同源建模、从头预测等）；
最终通过解析空间结构，完成功能注释、活性位点定位、分子相互作用解析，形成完整的一级结构 → 结构预测 → 功能解析研究范式。

## 2.9 氨基酸之间的相互作用（结构稳定核心）
线性多肽链能够折叠成稳定、紧凑、有功能的蛋白质，依赖于氨基酸残基之间的各类相互作用。这些作用分为共价相互作用和非共价相互作用两大类，共同构成蛋白质结构稳定的核心机制。

共价作用键能高、稳定性强，主要用于固定骨架与刚性交联：
1.  **肽键：**
连接氨基酸的基本共价键，构成多肽主链，是一级结构的化学基础。其部分双键性质与肽平面刚性，为高级结构提供了构象约束框架。

2.  **二硫键：**
由两个半胱氨酸侧链巯基（‑SH）氧化形成的共价交联键，主要稳定三级结构与四级结构，常见于胞外蛋白、分泌蛋白，可大幅提升蛋白质的耐热、耐酸碱与抗变性能力。

非共价作用较弱、可逆，是折叠驱动、构象调节与精细稳定的主力：
1.  **氢键：**
维持蛋白质二级结构的关键作用力。主链 C=O 与 N‑H 形成的氢键直接支撑 α‑螺旋、β‑折叠；侧链极性基团间的氢键进一步稳定三级结构，是结构规则性的重要来源。
2.  **疏水相互作用：**
蛋白质折叠的最主要驱动力。疏水 / 非极性侧链为避开水环境，向蛋白内部聚集形成疏水核心，由熵增驱动，是多肽链从线性变球状的根本原因

3.  **离子键（盐桥）：**
带正电的碱性氨基酸与带负电的酸性氨基酸之间的静电吸引作用。多分布在蛋白表面或结构界面，对稳定构象、调节溶解性与分子识别都有重要贡献。

4.  **范德华力：**
原子间近距离弱相互作用，单个作用极弱，但在蛋白质内部残基紧密堆积时，大量范德华力的协同效应显著提升结构致密性与稳定性，是维持精细空间构象的基础。

各类作用对稳定性的贡献清晰分工：
肽键与二硫键构建并交联主链骨架；
疏水相互作用决定整体折叠趋势；
氢键、离子键、范德华力共同精细稳定最终构象。
多种作用力协同，使蛋白质既能保持稳定结构，又能在执行功能时发生合理构象变化，实现结构与功能的统一。
